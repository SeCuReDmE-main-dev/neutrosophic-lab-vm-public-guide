# Maintainer And Researcher Notes

This lab is designed to make an applied research path visible, not to ask upstream maintainers to accept changes.

## For Maintainers

The lab branches are fork-only public builds. They do not imply upstream approval.

If you maintain one of the upstream projects, this repository is not asking you to review a PR or triage an issue.

## For Researchers

Use the evidence bundle to inspect:

- which repository was tested
- which branch was used
- which commit was used
- which model was used
- which routine was run
- when the artifact was generated

## Academic Framing

The lab explores how neutrosophic truth, indeterminacy, and falsity concepts can be represented in real AI software systems.
