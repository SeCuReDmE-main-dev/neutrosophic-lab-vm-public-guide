# What This Lab Is Not

This lab is not an upstream contribution channel.

It does not:

- request upstream maintainers to review this work
- claim upstream acceptance from any project
- push code to mainstream repositories
- provide general support or troubleshooting
- promise long-term maintenance
- copy secrets into a VM
- write scheduled reports into Windows user directories
- act as a general-purpose autonomous coding agent

## No PR Evaluation

This documentation repository is not for pull requests. PRs may be closed without review.

## No Issue Support

Issues are not treated as support tickets. If issues are enabled, they may be closed without response.

## No Product Claim

This lab is a research and reproducibility companion. It is not a commercial product, a supported platform, or a hosted service.

## No Adaptive Engine

The orchestrator workspace (`lab-orchestrator-public`) is not an adaptive engine. It orchestrates, collects, evaluates, and reports. It does not autonomously modify workspaces or write back into public repo source trees.
