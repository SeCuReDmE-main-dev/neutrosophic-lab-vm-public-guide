# What This Lab Is Not

This lab is not an upstream contribution channel.

It does not:

- request upstream maintainers to review this work
- claim upstream acceptance
- push code to mainstream repositories
- provide general support
- promise long-term maintenance
- copy secrets into a VM
- write scheduled reports into Windows user directories

## No PR Evaluation

This documentation repository is not for pull requests. PRs may be closed without review.

## No Issue Support

Issues are not treated as support tickets. If issues are enabled, they may be closed without response.

## No Product Claim

This lab is a research and reproducibility companion. It is not a commercial product or a supported platform.
