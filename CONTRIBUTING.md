# Contributing

Contributions are closed by default.

This repository exists to publish a public VM lab guide and reproducibility notes. It is not an open development queue.

## Pull Requests

Pull requests may be closed without review. Code changes are not accepted or maintained through this repository.

## Issues

Issues may be disabled or closed without response. This repository is not a technical support desk.

## Academic Corrections

If an academic citation or factual correction is needed, use external correspondence with the author rather than GitHub issues. This repository does not triage academic feedback through its issue tracker.

## Code Changes

Code changes are not accepted or maintained through this repository. The implementation source is `SeCuReDmE-main-dev/moteur-neutrosophique-adaptable` (branch `PaQBoT`).
